What is Bash?

Bash (**Bourne Again Shell**) is the default shell for most **Linux distributions**. It allows users to interact with the system using **commands and scripts**, making it a powerful tool for **automation and security testing**.

üêß Bash in Linux

Bash is deeply integrated into Linux. It is used for:

  • **Automating tasks** - Cron jobs, system maintenance, backups.
  • **System administration** - User management, log analysis, and permissions.
  • **Networking** - Managing servers, SSH automation, and packet capturing.
  • **Process management** - Monitoring system resources and managing services.

Common Bash Commands

Command Function
ls List directory contents
cat file.txt View the contents of a file

Automating a Network Scan

This **Bash script** automates a network scan using **Nmap**:

#!/bin/bash
echo "Enter target IP:"
read IP
nmap -sV -A $IP

Getting Started with Bash

Bash comes **pre-installed** on all Linux distributions. If you're using **Windows**, install **Git Bash** or **WSL (Windows Subsystem for Linux)**.
